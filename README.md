# Optimisation du TSP via ACO et Deep Learning
**M1 MIAGE MIXTE - Université Paris Nanterre**

## Description
Ce projet explore l'apport du Machine Learning (Réseaux de neurones) pour améliorer une métaheuristique (Colonies de fourmis - ACO) afin de résoudre le Problème du Voyageur de Commerce (TSP).

## Structure du Dépôt
- `src/` : Code source Python (ACO, Modèle ML, Hybridation).
- `data/` : Instances de test (villes).
- `results/` : Graphiques de performance.
